## Como estudiar este tema

Responda cada afirmación sin mirar la pauta y, si marca `Falso`, reescríbala de forma correcta. Esa segunda parte es la que realmente confirma si el concepto está aprendido.

### Ejercicio VII.1

En C++, si una variable es de tipo `int*`, entonces almacena directamente un entero.

### Ejercicio VII.2

La expresión `v[i]` en un arreglo es equivalente a `*(v + i)`.

### Ejercicio VII.3

Si una clase base tiene métodos virtuales, entonces su destructor no necesita ser virtual.

### Ejercicio VII.4

Una pila implementa la política FIFO.

### Ejercicio VII.5

Una cola implementa la política LIFO.

### Ejercicio VII.6

En una lista enlazada simple, acceder al último nodo siempre cuesta `O(1)`.

### Ejercicio VII.7

Un algoritmo `O(n)` siempre es más rápido que uno `O(log n)` para cualquier valor de `n`.

### Ejercicio VII.8

En una llamada por referencia (`Tipo&`), la función trabaja sobre el objeto original y no sobre una copia.

### Ejercicio VII.9

Si un método no es `virtual`, una llamada por puntero base puede ejecutar la versión de la clase derivada.

### Ejercicio VII.10

Si una función crea memoria con `new[]`, entonces debe liberarla con `delete[]`.

### Ejercicio VII.11

La operación `pop()` de una pila retorna y elimina el elemento del tope.

### Ejercicio VII.12

Una llamada a un método virtual por referencia base puede ejecutar una implementación derivada.

### Ejercicio VII.13

En una lista enlazada simple, insertar un nodo después de uno ya conocido cuesta `O(1)`.

### Ejercicio VII.14

La operación `front()` en una cola elimina el primer elemento.

### Ejercicio VII.15

Si se pasa un puntero por valor a una función, la función puede cambiar permanentemente hacia dónde apunta el puntero original.

### Ejercicio VII.16

`nullptr` representa la ausencia de dirección válida de memoria.

### Ejercicio VII.17

La búsqueda binaria puede aplicarse correctamente sobre datos no ordenados.

### Ejercicio VII.18

Una cola puede implementarse usando dos pilas.

### Ejercicio VII.19

El object slicing ocurre cuando un objeto derivado se copia por valor en una variable de tipo base.

### Ejercicio VII.20

Recorrer una lista enlazada completa una sola vez tiene complejidad temporal `O(n)`.
